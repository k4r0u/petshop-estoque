## Projeto universitário: Petcare - Sistema de controle de estoque 

Foi desenvolvida uma aplicação web simples e funcional para gerenciamento de estoque de um petshop fictício, utilizando Java com o framework Spring Boot.
O sistema oferece um CRUD (Create, Read, Update, Delete) completo para produtos, com autenticação mockada via Spring Security (usuário e senha fixos), cadastro, listagem, visualização, edição e exclusão de itens contendo nome, descrição, quantidade em estoque e preço.

No futuro pretendo melhorar o sistema por desafio pessoal :)
